# 구현 계획

## 개요

시도별·시군구별·월별 보행자 사고 데이터 기반 교통 안전 분석 대시보드를 구현합니다. 복잡한 기능보다는 핵심 가치를 제공하는 세 가지 주요 기능에 집중합니다: 1) 횡단보도 + 신호등 시각화, 2) 개선된 팝업, 3) 사고 히트맵. 프론트엔드에서 실시간으로 안전/위험 지표를 계산하여 백엔드 복잡성을 최소화합니다.
